# Практическая работа №10 Вариант 1 (Шифр Цезаря)

## Цель практической работы
Закрепить практические навыки разработки, отладки, рефакторинга, ручного и автоматизированного тестирования графического приложения с использованием техники «Разработка через тестирование» (Test-Driven Development, TDD).

## Описание проекта
Данное приложение реализует класс `CaesarCipher`, предназначенный для шифрования и дешифрования текста с использованием метода Цезаря. Поддерживаются английский и русский алфавиты, а также цифры. Остальные символы остаются без изменений.
